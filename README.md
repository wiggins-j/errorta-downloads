# Errorta Downloads

This repository hosts the public download page and release assets for
Errorta, the local AI desktop app that admits when it is wrong and
remembers your corrections.

## Download Page

The public page is <https://errorta.app>.

Installer builds and supporting artifacts are published on the
[GitHub Releases page](https://github.com/wiggins-j/%65rrorta%2Ddownloads/releases).
Release notes include SHA-256 checksums for files that should be
verified after download.

## Built On AIAR

Errorta is built on [AIAR](https://github.com/wiggins-j/aiar), the
open-source local-RAG framework that provides retrieval, LLM-as-judge
evaluation, and grounding/correction primitives.
